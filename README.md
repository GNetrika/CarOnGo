A Full-Stack Java Web Application using Servlets, JSP, PostgreSQL & Apache Tomcat 9

CarOnGo is a complete car rental management system built using Java Servlets, JSP, MVC architecture, PostgreSQL.

It allows customers to browse available cars, book a rental, and view booking confirmation.
Admins can add cars, view all cars, and view bookings.

This project is fully developed and deployed in Eclipse IDE with Apache Tomcat 9.0.
